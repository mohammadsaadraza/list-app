# list-app
A simple interface to maintain lists

  1. Enter a list name
  2. If the name exists in DB, the list contents will be displayed. Else a new list is created with that name
  3. Add a new item to the list, or remove current items
  
## Instructions to use

  1. In the `/config/keys.js`, put in the MongoDB Connect URI of your own. 
  2. In root folder run command `npm run devServer` to start up the backend Node Server.
  3. In `/client` folder run command `npm start` to start the frontend React Server.
  4. Now you are ready to use the app.
